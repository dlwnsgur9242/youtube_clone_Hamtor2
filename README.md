# 커밋 메세지 룰
    Line 1: 제목
            - '태그(레이어): 기능' 으로 입력
            - 예: Feat(H): Sidebar

            # 태그
            - 명령형 동사 사용. 대문자로 시작
            - 예: Init (첫 업로드), Feat (새로운 기능), Fix (수정), Remove (삭제)

            # 레이어
            - H (Home 페이지) / C (Channel 페이지) / V (Video 페이지)

            # 기능
            - 사용한 변수명 또는 함수명 입력

    Line 2: 공란

    Line 3: 본문
            - What, Why 등 작성
